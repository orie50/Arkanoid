1) ביטוי שמעלים בחזקת ביטוי שגם הוא מועלה בחזקה הפישוט שלו הוא הביטוי הראשון בחזקת הביטוי השני כפול השלישי. לדוגמא:
(x^y)^z => x^(y*z) שיטה: נבדוק אם המעריך הוא מסוג ביטוי של חזקה ואם כן נחזיר את הראשון בחזקת כפל של השני והשלישי.

2) זהות טריגונומטרית: cos(90 + x) => - sin(x) שיטה: נחזיר את הארגומנטים בצורת new Neg(new Sin(arg)).

3) זהות טריגונומטרית: sin(90 + x) => cos(x) שיטה: נחזיר את הארגומנטים בצורת new Cos(arg).

4) צמצום של המינוסים כאשר יש יותר ממינוס אחד כל שני מינוסים רצופים מבטלים זה את זה (שלילה כפולה) לדוגמא:
-(-(-(x +y) => -(x +y) שיטה: נבדוק אם הארגומנט של הביטוי הוא neg ואם כן נחזיר את הארגומנט של הארגומנט כדי לבטל את השלילה הכפולה.

5) ביטוי בחזקת 0 = 1 לדוגמא: x^0 => 1 שיטה: נבדוק אם argB הוא אפס ואם כן נחזיר new Num(1).

6) ביטוי בחזקת 1 = ביטוי עצמו לדוגמא: x^1 => x שיטה: נבדוק אם argB הוא אחד ואם כן נחזיר argA.


7) חילוק של ביטוי ב1 = ביטוי עצמו לדוגמא: (x +y)/1 => x+yשיטה: נבדוק אם argB הוא אחד ואם כן נחזיר argA.
